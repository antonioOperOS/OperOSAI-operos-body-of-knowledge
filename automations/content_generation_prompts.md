# Content Generation Prompts

## Purpose

Standardize how OperOS source objects become polished outputs without losing consistency.

## Framework to digital-book page

Use the framework source object. Create a clear digital-book page with executive framing, the canonical visual, practical steps, examples, metrics, failure modes, AI augmentation, and implementation path. Preserve the object ID and version.

## Framework to sales one-pager

Create an executive one-pager that explains the pain, the cost of inaction, the OperOS insight, the visual model, the before/after transformation, expected outcomes, and the next action.

## Framework to workshop module

Create a 60-90 minute workshop with facilitator instructions, participant worksheet, timed agenda, exercises, outputs, and follow-up actions.

## Framework to AI coach object

Convert the framework into retrieval-ready knowledge: concise definition, when to use, diagnostic questions, recommended actions, required data, escalation rules, and linked objects.

## Guardrails

- Do not copy protected source text.
- Use original OperOS language.
- Preserve object metadata.
- Link dependencies.
- Produce measurable outputs.
- End with OperOS implementation path.
